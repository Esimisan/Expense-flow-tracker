ExpenseFlow is a responsive personal finance web app built with vanilla HTML, CSS and JavaScript, no frameworks, no backend. Live demo:  https://esimisan.github.io/Expense-flow-tracker/

Users can register with their name and email, and every page is protected with an auth guard. The dashboard shows a live balance with total income and expenses, lets you add transactions across 10 categories, and warns you when you exceed your monthly budget. The analysis page breaks down spending month by month with an animated doughnut chart, category progress bars, and a full transaction list. The settings page lets you pick an avatar, choose from 80+ currencies, set a monthly budget, and toggle dark mode across all pages instantly.

Built with HTML5, CSS3, vanilla JavaScript, localStorage, Chart.js and Font Awesome. Fully responsive using vw, rem and em — no media queries needed. All data lives in localStorage so no backend is required, though clearing browser data or switching devices will reset the app. Currency switching updates the symbol only with no conversion applied.
